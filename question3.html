Review 3 different GUI Git clients

Since git is a powerfull tool for devellopers it has attracted attation of the community and different GUI clients has been developed since to suit the need of different users.

Sourcetree

It is a very easy git client to learn and offer a simplify distributed version control. Give expert the ability to review changesets, stash, easy branching and enable a clear view of your code.

TortoiseGit

TortoiseGit is an open source software. Give a freedom of use with any type of development tools available. It mainly uses the context menu of the window explorer

Tower

Tower is one of the best Git client for Mac and Windows, it is largely used by companies such as Apple, Google, Amazon, eBay and more
